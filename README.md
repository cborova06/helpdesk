## BRV Helpdesk

BRV Helpdesk; müşteri taleplerini toplamak, ekipler arasında yönlendirmek ve bilgi bankasıyla hızlı çözüm sunmak için geliştirilmiş, BRV Software ürünü bir destek ve etkileşim platformudur.

- Tek noktadan talep yönetimi (e‑posta/portal)
- Esnek SLA ve atama kuralları
- Bilgi bankası ve hazır yanıtlara dayalı hızlı yanıt
- Kullanışlı arayüz, kısayollar ve toplu işlemler

Daha fazla bilgi: https://www.brvsoftware.com.tr/

## Hızlı Başlangıç (Özet)
Aşağıdaki adımlar temel kurulumu özetler. Kurulum/dokümantasyon ihtiyacınız için bizimle iletişime geçebilirsiniz.

1) Depoyu alın ve kurulum ortamınızı hazırlayın.
2) Uygulamayı sitenize ekleyin ve derleyin.
3) Sistem ayarlarınızı (ekip, SLA, e‑posta vb.) yapılandırın.

Örnek komutlar (geliştirici ortamı):
```
bench get-app https://github.com/cborova06/helpdesk --branch main
bench --site your-site install-app helpdesk
bench build && bench restart
```
